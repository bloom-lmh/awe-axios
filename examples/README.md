# Examples

This folder contains small reference snippets for the rebuilt package layout.

- `core-basic.ts`: basic HTTP decorators
- `core-strategies.ts`: retry, debounce, and throttle decorators
- `mock-basic.ts`: MSW-backed mocking
- `mock-ws-protocol.ts`: decorator-first WebSocket protocol mocking with state, guards, and ack/error envelopes
- `ioc-aop-basic.ts`: dependency injection and aspect weaving
- `umbrella-imports.ts`: import styles for the `decoraxios` umbrella package, including HTTP, WebSocket mock, and IoC/AOP APIs
