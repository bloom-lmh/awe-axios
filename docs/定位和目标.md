# AxiosPlus

[[toc]]

## 定位和目标

AxiosPlus 是一个基于 Axios 的扩展库，整合 Axios、MSW 和 FackerJS 等库的功能，旨在为 Axios 提供更多的功能和便利

## 设计理念

1. 注解驱动：通过注解的方式来驱动 AxiosPlus 的功能，简化配置，提高开发效率。
2. IOC 和 DI：AxiosPlus 使用依赖注入的方式来管理接口和使用接口
3. 无侵入：AxiosPlus 不会修改 这些库的任何原有功能，只会在 这些库的的基础上进行扩展和封装。
4. 灵活性：支持自定义注解，可以灵活地扩展 AxiosPlus 的功能。
5. 可配置：AxiosPlus 提供丰富的配置项，可以灵活地通过配置 AxiosPlus 的来改变 AxiosPlus 的行为。并采用配置优先级和配置合并
