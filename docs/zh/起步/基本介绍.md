# 基本介绍

## 什么是 AxiosPlus？

`AxiosPlus` 是在 `axios` 基础上进行扩展的增强型 `HTTP` 请求工具库。它通过装饰器模式、配置扩展等方式，为 `axios` 提供了更丰富的功能和更灵活的使用方式，同时保持了与 `axios` 生态的兼容性。

核心特性包括：

- 基于装饰器的 `API` 定义方式，简化请求配置
- 内置请求重传（`Retry`）、防抖（`Debounce`）、节流（`Throttle`）机制
- 支持自定义 `axios` 实例管理（`RefAxios`）
- 提供请求/响应数据转换（`Transform`）能力
- 集成 `mock` 功能，便于开发调试
- 完善的 `TypeScript` 类型定义，提供良好的类型提示

## 使用场景

`AxiosPlus` 适用于需要处理各类 `HTTP` 请求的前端应用场景，尤其适合：

1. **企业级应用开发**：通过装饰器集中管理 API 配置，提高代码可维护性
2. **高频请求场景**：利用防抖、节流功能优化请求性能，减少不必要的网络请求
3. **不稳定网络环境**：通过请求重传机制提升请求成功率
4. **多环境适配**：支持自定义 axios 实例，可同时适配不同域名、认证方式的后端服务
5. **前后端并行开发**：内置 mock 功能，无需等待后端接口就绪即可进行开发
6. **需要数据转换的场景**：提供便捷的请求/响应数据转换能力，处理数据格式转换、加密解密等需求

无论是 `Vue、React` 等框架应用，还是原生 `JavaScript` 项目，都可以通过 `AxiosPlus` 简化 `HTTP` 请求的处理逻辑。
